# Accessibility Checklist

ใช้ checklist นี้ตรวจหน้าเว็บหนังสือและ UI example ถ้ามี

- [ ] heading เรียงลำดับเหมาะสม
- [ ] link text อ่านแล้วเข้าใจปลายทาง
- [ ] code block อ่านได้บน mobile
- [ ] contrast เพียงพอ
- [ ] keyboard navigation ใช้งานได้
- [ ] form controls มี label ถ้ามี UI
- [ ] error messages ไม่พึ่งสีอย่างเดียว
- [ ] images มี alt text ถ้าจำเป็นต่อความเข้าใจ
